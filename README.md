# CaosHub (Update) - v1.1.16!
O **CaosHub** é um script de automação e utilitários desenvolvido para a plataforma Roblox, focado em oferecer uma interface intuitiva e ferramentas poderosas para melhorar a experiência do usuário. Utilizando a biblioteca **Rayfield**, o hub proporciona uma navegação fluida e visualmente moderna.

## ✨ Novidades da Versão 1.1.16!
* **Aba de Ferramentas:** Uma nova seção dedicada a utilitários de sistema.
* **Seletor de Objetos Integrado:** Agora é possível selecionar e listar objetos do mapa apenas clicando neles com o mouse.

## 🚀 Funcionalidades Principais!

### 🏃 Jogador (Movimentação):
* **Ajuste de Velocidade (WalkSpeed):** Altere a velocidade de caminhada em tempo real.
* **Noclip:** Atravesse paredes e obstáculos facilmente.
* **Fly (Voo):** Explore o mapa de qualquer ângulo com controles de direção baseados na câmera.
* **Pulo Infinito:** Remova a limitação de saltos do personagem.

### 🛠️ Ferramentas (Seletor):
* **Seleção por Clique:** Ative o seletor e clique em qualquer parte do mapa para identificá-la.
* **Feedback Visual:** Objetos selecionados recebem um efeito de *Highlight* (brilho).
* **Gestão de Lista:** Visualize todos os itens selecionados em uma lista dinâmica ou limpe tudo com um clique (ou tecla de atalho `C`).

### 📍 Teleporte:
* **Busca Inteligente:** Digite o nome ou ID de um jogador para localizá-lo.
* **Teleporte Instantâneo:** Vá direto para a posição de qualquer player no servidor com sistema de notificação de status.

## 🛠️ Como Usar:
Para executar o script, utilize seu executor de preferência e insira o código fonte disponível neste repositório.
